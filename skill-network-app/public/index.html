Months 1-2: The Bedrock

Mastering core analytical concepts, advanced Excel, and the fundamentals of SQL.

  • Data Analysis Process
  • Advanced Excel: VLOOKUP/XLOOKUP, Pivot Tables, Power Query
  • Business Metrics & KPIs
  • SELECT, FROM, WHERE
  • JOINs (INNER, LEFT, RIGHT)
  • GROUP BY & Aggregations
  • Basic Subqueries
  • Use Gemini as a Socratic learning tutor for SQL
  • Leverage Excel's AI-powered "Analyze Data" feature

Months 3-4: Storytelling & Programming

Learning to visualize data with BI tools and automating analysis with Python.

  • Choose Power BI or Tableau
  • Principles of effective visualization
  • Build interactive, multi-page dashboards
  • Pandas for data manipulation & cleaning
  • NumPy for numerical operations
  • Matplotlib & Seaborn for plotting
  • Use GitHub Copilot to accelerate Python coding
  • Debug code and errors with Gemini
  • Use Power BI's natural language Q&A feature

Months 5-6: Advanced Topics

Diving into advanced SQL, statistics, and foundational machine learning concepts.

  • SQL Window Functions & CTEs
  • Practical Statistics: Hypothesis Testing, A/B testing analysis
  • Correlation vs. Causation
  • Supervised vs. Unsupervised Learning
  • Concepts: Regression, Classification, Clustering
  • Build a simple model with Scikit-learn
  • Use AI to run statistical tests
  • Explore no-code AutoML tools in Power BI or GCP

Months 7-8: The Launchpad

Focusing on a capstone project, cloud fundamentals, and career preparation.

  • Basics of AWS, GCP, or Azure
  • Understand cloud data storage (e.g., S3, BigQuery)

An end-to-end project combining SQL, Python, and a BI tool to solve a real-world problem. This is your portfolio centerpiece.

  • Build an online portfolio on GitHub/Kaggle
  • Use AI to optimize your resume and LinkedIn for ATS
  • Practice interviews with AI-generated questions

Month 1: Excel for Data Analysis

Building a strong foundation with the most common data tool.

  • Advanced Formulas (XLOOKUP, INDEX-MATCH)
  • Data Cleaning & Formatting
  • Pivot Tables & Pivot Charts

Analyze a sales dataset to build a summary dashboard with slicers in Excel.

Month 2: Querying with SQL

Learning to talk to databases, a non-negotiable skill.

  • Retrieving data with SELECT, WHERE
  • Combining tables with JOINs
  • Aggregating data with GROUP BY

Answer 10 business questions about a customer dataset using only SQL queries.

Month 3: Visualizing with BI Tools

Telling compelling stories with data dashboards.

  • Choose Power BI or Tableau
  • Connect to data sources (Excel, SQL)
  • Create interactive dashboards with filters

Build a one-page, interactive sales dashboard in your chosen BI tool.

Month 4: Analyzing with Python

Using code to automate cleaning and analysis tasks.

  • Python Pandas for data cleaning
  • Matplotlib & Seaborn for basic plotting
  • Perform Exploratory Data Analysis (EDA)

Use Python to clean a messy dataset and create a short report with key findings and charts.

Month 5: Practical Statistics

Understanding the "why" behind the numbers to make sound decisions.

  • Descriptive Statistics (Mean, Median, Mode)
  • Understanding distributions
  • Correlation vs. Causation

Perform a statistical analysis on a dataset and write a summary of your findings about the relationships between variables.

Month 6: Capstone & Career Prep

Combining all skills into a portfolio project and preparing for the job market.

Analyze a complex dataset from end-to-end. Define a business problem, use SQL/Python to analyze, and present findings in a BI dashboard.

  • Build a simple online portfolio
  • Craft a data-focused resume
  • Practice common interview questions
